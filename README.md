# 📚 Repositório de Estudos em Java

Este repositório é um espaço onde eu organizo tudo que venho aprendendo em Java, tanto em estudos pessoais quanto acadêmicos.

## 🚀 Objetivo

Consolidar meu aprendizado em Java através da prática, registrando:

- Exercícios
- Projetos
- Testes de conceitos
- Experimentos com código

A ideia é evoluir constantemente e acompanhar minha evolução como desenvolvedor.

## 🧠 O que você vai encontrar aqui

- Lógica de programação
- Programação Orientada a Objetos (POO)
- Estruturas de dados
- Pequenos projetos
- Integrações (ex: banco de dados, APIs)
- Testes e experimentos

## 📁 Organização

O repositório pode conter diferentes pastas, como:

- `Aulas/` → conteúdos e exercícios da faculdade
- `Projetos/` → projetos maiores ou completos
- `mvc/` → testes com arquitetura MVC
- Outros diretórios conforme evolução dos estudos

## 🛠️ Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git & GitHub

## 📈 Evolução

Este repositório está em constante atualização conforme avanço nos estudos.

## 📌 Observação

Este é um ambiente de aprendizado, então é normal encontrar códigos simples, experimentais ou em evolução.
